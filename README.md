## Особенности

1) Проект устойчив к смене фреймворков для работы с 3d моделями и с разными апи
2) Проект способен оптимально использовать сразу несколько фреймворков для работы с 3d моделями
3) Вся функциональность связанная с работой с 3d моделями и ии вынесена в отдельные модуль разбитый согласно методологии `FSD`

## Архитектура

1) Проект подчиняется правилам `FSD`
2) для обеспечения особенностей введены дополнительные правила по использованию слайсов
    - в наименовании слайсов при исполльзовании определенного фреймворка или апи необходимо добавлять расширение `.framework`
    - при условии, что в слайсах, использующих разные фреймворки есть общие компоненты, необходимо вынести общую функциональность на слой ниже и назвать аналогично слайсам но без расширения `.framework`
